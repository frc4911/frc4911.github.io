    WPILib (required)

    What is it?

    WPILib is the standard software library for FRC robots. It supports Java, C++, and Python.

    Why is it useful?

    WPILib's core purpose is to enable teams to program their robots effectively. It provides a software framework that simplifies the process of controlling motors, sensors, and other robot hardware. By abstracting away complex hardware interactions, WPILib allows teams to concentrate on developing sophisticated robot strategies and behaviors.

    How do I get it?

    WPILib releases page.

    Github Desktop (required)

    What is it?

    GitHub Desktop is a user-friendly application that provides a visual interface for Git.

    Why is it useful?

    Github Desktop simplifies interactions with GitHub repositories. It allows users to perform common Git tasks like committing, branching, and pushing changes without using the command line. Designed for ease of use, it makes Git more accessible, especially for those new to version control.

    How do I get it?

    Github Desktop Page.

    FRC Game Tools (recommended)

    What is it?

    The FRC Game Tools is a software package provided by National Instruments (NI) that is essential for teams participating in FRC.

    Why is it useful?

    It includes the FRC Driver Station, which allows teams to control their robots during matches, and the roboRIO Imaging Tool, used to configure the robot's onboard computer. These tools are essential for managing robot communication and operation during competitions.

    How do I get it?

    FRC Game Tools page

    AdvantageScope (recommended)

    What is it?

    AdvantageScope is a robot diagnostics, log review/analysis, and data visualization application.

    Why is it useful?

    It reads logs in WPILOG, DS log, Hoot (CTRE), and RLOG file formats, plus live robot data viewing using NT4 or RLOG streaming. AdvantageScope can be used with any WPILib project, but is also optimized for use with our AdvantageKit log replay framework.

    How do I get it?

    AdvantageScope releases page

    Elastic (recommended)

    What is it?

    Elastic is a simple and modern dashboard for FRC.

    Why is it useful?

    It was built with the goal of being a lightweight drag & drop dashboard with an easy setup and elegant UI designed for a high pressure competition environment.

    How do I get it?

    Elastic releases page

    Phoenix Tuner X (recommended)

    What is it?

    Phoenix Tuner X is a software application from CTR Electronics that simplifies the configuration and management of their hardware.

    Why is it useful?

    Essentially, it allows teams to update firmware, set device IDs, monitor performance, and control motors through a user-friendly interface. This tool streamlines the process of optimizing robot hardware, making it easier for teams to fine-tune their robot's performance.

    How do I get it?

    Phoenix Tuner X page

    Rev Hardware Client (recommended)

    What is it?

    The REV Hardware Client is a software application developed by REV Robotics designed to simplify the management and updating of REV Robotics hardware.

    Why is it useful?

    It allows users to easily update firmware, configure devices like the Control Hub and SPARK MAX motor controllers, and perform diagnostics. This tool streamlines hardware maintenance, making it easier for teams to keep their robots running smoothly.

    How do I get it?

    Rev Software releases page

    Choreo (recommended)

    What is it?

    Choreo is a graphical tool for planning time-optimized trajectories for autonomous mobile robots.

    Why is it useful?

    It is a tool for transforming 2D paths from rough sketches into optimal trajectories. With minimum-travel-time paths that respect your robot’s physical properties and limits, you can have a high degree of trust that your robot will be able to follow Choreo’s paths on the first try.

    How do I get it?

    Choreo releases page
